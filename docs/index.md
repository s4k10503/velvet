# Velvet API Reference

**Velvet** is a React-style declarative UI framework for Unity UI Toolkit — a Virtual DOM,
reconciler, hooks, a Zustand-style store, and utility-first styling, all driven from C#.

- [API Reference](xref:Velvet) — generated from the `Velvet` runtime assembly's XML documentation.
- For installation, getting started, and guides, see the repository README and the package `Documentation~/` folder.

> This site documents the public runtime API surface (the `Velvet` namespace). Editor-only and
> internal types are omitted.
